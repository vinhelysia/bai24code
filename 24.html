Sơ đồ Bài 24: Xâu ký tự (Strings)

Hãy bấm vào các từ có nét đứt để xem cách máy tính hiểu và xử lý xâu ký tự.

# 1. Khởi tạo và đếm số lượng ký tự
s = "Tin học 11"
so_ky_tu = len(s)
print("Chiều dài của xâu:", so_ky_tu) # Kết quả: 10

# 2. Truy cập bằng chỉ số (Index) và Tính bất biến
print("Ký tự đầu tiên:", s[0]) # Kết quả: 'T'
# s[0] = "t" <-- LỖI TYPE ERROR! Xâu không cho phép sửa từng ký tự

# 3. Duyệt xâu bằng chỉ số (khi cần biết vị trí)
for i in range(len(s)):
    print(s[i], end="-")

print()

# 4. Duyệt trực tiếp từng ký tự (nhanh, gọn)
for ch in s:
    print(ch, end="*")


# 5. Kiểm tra xâu con
if "học" in s:
    print("\nTìm thấy từ 'học'!")

Bấm vào một cấu trúc ở đoạn code trên để xem máy tính đang xử lý xâu ký tự như thế nào.
